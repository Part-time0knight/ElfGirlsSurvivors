# ElfGirlsSurvivors

Тестовое задание для Saber Interactive

## Стэк
- **Unity**
- **C#**
- **Zenject**
- **DOTween**
- **UniTask**
- **UniRx**

## Архитектурные решения

### Игровой процесс
- Используется **конечный автомат (Finite State Machine, FSM)**
- Обмен данными инкапсулирован через DI-контексты:
  - Сервисы определяются на уровне сцены
  - Логика, специфичная для объекта, находится в его локальном контексте

### Игровые объекты
- Применяется **паттерн «Фасад» (Facade)** для управления сложными сущностями (например, корабль игрока)
- Точка входа фасада делегирует управление **конечному автомату (FSM)**, который координирует внутренние компоненты

### UI
- Реализован с использованием паттерна **MVVM (Model-View-ViewModel)**
- Для управления экранами применяется отдельный конечный автомат
